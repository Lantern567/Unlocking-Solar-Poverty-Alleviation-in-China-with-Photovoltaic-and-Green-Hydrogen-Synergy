# 经济性指标计算模型

## 1. LCOE（平准化度电成本）

### 计算公式
```math
\text{LCOE} = \frac{\sum_{t=0}^{N} \frac{C_t}{(1+i)^t}}{\sum_{t=0}^{N} \frac{E_t}{(1+i)^t}}
```

### 参数说明
- $C_t$：第$t$年总成本（元）
- $E_t$：第$t$年发电量（kWh）
- $i$：贴现率
- $N$：项目寿命周期（年）

### 解释
LCOE表示在整个项目生命周期内，单位电量的平均成本，考虑了所有投资、运维等费用的现值。

---

## 2. NPV（净现值）

### 计算公式
```math
\text{NPV} = \sum_{t=0}^{N} \frac{C_t}{(1+i)^t}
```

### 参数说明
- $C_t$：第$t$年净现金流（元）
- $i$：贴现率
- $N$：项目寿命周期（年）

### 解释
NPV用于衡量项目在考虑资金时间价值后的总收益，NPV大于0表示项目可行。

---

## 3. IRR（内部收益率）

### 计算模型
IRR是使NPV为0的贴现率$i^*$，即：
```math
0 = \sum_{t=0}^{N} \frac{C_t}{(1+i^*)^t}
```

### 参数说明
- $C_t$：第$t$年净现金流（元）
- $N$：项目寿命周期（年）
- $i^*$：使NPV为0的贴现率（即IRR）

### 解释
IRR反映了项目的实际收益能力，IRR大于资本成本率时项目可行。

---

## 4. ROI（投资回报率）

### 计算公式
```math
\text{ROI} = \frac{\text{总收益} - \text{总投资}}{\text{总投资}}
```
或
```math
\text{ROI} = \frac{\sum_{t=1}^{N} E_t \cdot P - \text{总投资}}{\text{总投资}}
```

### 参数说明
- $E_t$：第$t$年发电量（kWh）
- $P$：电价（元/kWh）
- 总投资：初始投资+全周期成本（元）
- $N$：项目寿命周期（年）

### 解释
ROI衡量投资带来的收益水平，ROI越高，投资回报越好。
