# 光伏储能系统容量与成本计算方法

## 1. 储能容量计算

### 计算公式
```math
E_{\text{storage}} = \frac{E_{\text{year}}}{365} \times \beta \times 1000 \times \frac{1.2 + 0.8}{2}
```

### 参数说明
- $E_{\text{year}}$：年发电量（单位：kWh/年）
- $\beta$：弃电率（Curtailed Rate，无量纲）
- 计算步骤：
  1. 年发电量转换为日均发电量（$E_{\text{year}}/365$）
  2. 乘以弃电率$\beta$得到需要储存的电量
  3. 乘以1000进行单位转换
  4. 乘以$(1.2 + 0.8)/2 = 1.0$作为系数

## 2. 成本计算

### 初始投资成本
```math
C_{\text{init}} = C_{\text{PV}} + E_{\text{storage}} + C_{\text{ES}}
```
其中：
- $C_{\text{PV}}$：光伏系统成本（元/kW）
- $E_{\text{storage}}$：储能容量（kWh）
- $C_{\text{ES}}$：储能系统成本（元/kW）

### 年度运维成本
```math
C_{\text{annual}} = (O_{\text{PV}} + O_{\text{ES}} + C_{\text{F}} + C_{\text{tax}} + C_{\text{d}} + C_{\text{fix,storage}}) \times N
```
其中：
- $O_{\text{PV}}$：光伏系统运维成本（元/kW·年）
- $O_{\text{ES}}$：储能系统运维成本（元/kW·年）
- $C_{\text{F}}$：其他固定成本（元/kW·年）
- $C_{\text{tax}}$：税收成本（元/kW·年）
- $C_{\text{d}}$：折旧成本（元/kW·年）
- $C_{\text{fix,storage}}$：储能运维成本（元/kW·年）
- $N$：资产寿命（年）

### 总成本
```math
C_{\text{total}} = C_{\text{init}} + C_{\text{annual}}
```

## 3. 单位储能成本计算

### 计算公式
```math
C_{\text{storage,unit}} = \frac{E_{\text{storage}}}{E_{n} \times \beta}
```
其中：
- $E_{n}$：年发电量（考虑限电率后的实际发电量，kWh/年）
- $\beta$：弃电率（无量纲）
